# 🧬🤖DiabAI – Intelligent Diabetes Risk Prediction & Personal Health Assistant

🧬 Diabetes is one of the fastest-growing global health challenges. Early detection can change lives — and that’s exactly what my project aims to do.

💡 DiabAI – Your Diabetes Diagnosis Guidance with Personal Chatbot Assistant - A smart, interactive web platform designed to predict the likelihood of diabetes and provide personalized health guidance in a simple, accessible way.

🔹 What I Built & Achieved:
📌Designed a clean, user-friendly interface with HTML & CSS for effortless data entry.

📌Developed a Python Flask backend integrating machine learning models - SVM, KNN, Logistic Regression, Decision Tree, and Random Forest

📌Created a rule-based chatbot assistant to explain prediction outcomes and deliver tailored health tips.

🔹 Tech Stack & Skills Gained:

 🔧Python| Flask | HTML | CSS | Machine Learning | Data Visualization | Data Cleaning | Problem-Solving | Real World Applications

💙 This project taught me how to bridge technical expertise with real-world healthcare impact, combining my passion for data science with a mission to make health insights more accessible.
